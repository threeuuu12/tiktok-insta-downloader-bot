# بوت التحميل 🚀

**بوت تليجرام لتحميل الفيديوهات والميديا من مختلف المنصات**

بوت التحميل هو بوت تليجرام يتيح لك تحميل الفيديوهات والصور والملفات من:
- يوتيوب (YouTube)
- إنستغرام (Instagram)
- تيك توك (TikTok)
- تويتر / إكس (Twitter/X)
- فيسبوك (Facebook)
- وأي منصة أخرى مدعومة

كل ما عليك فعله هو إرسال الرابط إلى البوت، وسيقوم بتحميل المحتوى وإرساله إليك مباشرة داخل تليجرام.

## ✨ المميزات

- دعم واسع للمنصات الاجتماعية
- تحميل فيديو بجودة عالية (حسب التوفر)
- تحميل الصوت فقط (Audio)
- دعم التحميل بدون علامة مائية (في بعض المنصات)
- واجهة عربية وإنجليزية
- سريع وخفيف
- سهل التثبيت والنشر (Docker + بدون Docker)

## 🛠 التقنيات المستخدمة

- Python
- [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) أو Pyrogram
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) (لتحميل الفيديوهات)
- FFmpeg (لمعالجة الميديا)

## 🚀 طريقة الاستخدام

1. ابحث عن البوت في تليجرام (مثال: `@YourDownloadBot`)
2. اضغط **ابدأ** أو أرسل `/start`
3. أرسل رابط الفيديو أو الصورة المراد تحميلها
4. انتظر قليلاً وسيرسل لك البوت الملف
